# Getting Started

Welcome from [[Welcome]]! A few things to try:

1. **Wikilinks** — type `[[` and pick a note. Click a link to navigate; click a
   link to a note that doesn't exist yet to create it.
2. **Live Preview** — markdown renders inline as you type; the raw source shows
   only on the line your cursor is on.
3. **Backlinks** — the right sidebar lists every note that links here.

See also [[Nested Note]].
