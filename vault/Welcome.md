# Welcome

This is your vault. Notes are plain `.md` files on disk — the backend reads and
writes them directly, just like Obsidian.

## Try it out

- Open [[Getting Started]] to learn the basics.
- Notes can live in folders, e.g. [[Nested Note]].
- You can give a link a different label: [[Getting Started|start here]].

Tags work too: #welcome #demo

> Edits autosave to disk and sync live across tabs.
